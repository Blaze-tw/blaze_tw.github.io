🌐 **Сайт доступен:** https://blaze-tw.github.io/

## 🚀 Как начать использовать

### Шаг 1: Откройте сайт
Переходите по ссылке: **https://blaze-tw.github.io/**

### Шаг 2: Вход в админ панель
1. Нажмите на кнопку 🔓 в нижнем правом углу
2. Введите код доступа: **`любовь`** (в нижнем регистре)
3. Нажмите "Вход"

### Шаг 3: Загруженсие фотографий
1. В админ панели выберите "Загрузить фото"
2. Нажмите на область с надписью "Выберите фото" или перетащите изображения
3. Подтвердите загрузку нажав "Загрузить"
4. Фото появится в галерее на главной странице

### Шаг 4: Просмотр на других устройствах
1. Откройте сайт на другом устройстве/браузере
2. Перейдите в галерею (раздел "Наши Моменты")
3. Загруженные фото появятся автоматически через 5-10 секунд

## 📊 Статус синхронизации

В навигационной панели вверху сайта отображается:
- **🔄 Синхронизация...** - идет процесс синхронизации
- **✓ Синхронизировано (время)** - последняя успешная синхронизация
- **✗ Ошибка синхронизации** - проблема с подключением

## 💾 Хранение данных

### Локальное хранилище (браузер)
- Ключ: `anna_photos`
- Хранит: Все загруженные фото в формате Base64
- Максимальный размер: ~5MB (в зависимости от браузера)

### Облачное хранилище (GitHub)
- Файл: `anna-photos-data.json`
- Хранит: Резервная копия для синхронизации
- Автоматически обновляется при загрузке новых фото

## 🐛 Решение проблем

### Фото не синхронизируются
1. Проверьте подключение к интернету
2. Очистите кэш браузера (Ctrl+Shift+Delete)
3. Перезагрузите страницу (F5)
4. Откройте консоль (F12) и проверьте ошибки

### Админ панель не открывается
1. Убедитесь, что JavaScript включен
2. Попробуйте другой браузер
3. Убедитесь, что вводите код в нижнем регистре: `любовь`

### Фото не загружаются
1. Проверьте формат файла (JPG, PNG, GIF, WebP)
2. Проверьте размер файла (максимум 5MB)
3. Попробуйте другой браузер
4. Освободите место в памяти браузера
