# 📡 Руководство по синхронизации фото

## Как работает синхронизация?

Сайт использует **облачную синхронизацию** для обмена фото между устройствами через GitHub.

### Архитектура синхронизации

```
┌─────────────────────────────────────────────────────────────┐
│                        GitHub Pages                         │
│                   (blaze-tw.github.io)                      │
└────────────────────────┬────────────────────────────────────┘
                         │
         ┌───────────────┼───────────────┐
         │               │               │
    ┌────▼─────┐    ┌────▼─────┐   ┌────▼─────┐
    │Устройство│    │Устройство│   │Устройство│
    │    1     │    │    2     │   │    3     │
    │(ПК)      │    │(Телефон) │   │(Планшет) │
    └──────────┘    └──────────┘   └──────────┘
         │               │              │
         └───────────────┼──────────────┘
                         │
              Локальное хранилище + GitHub
```

## Процесс синхронизации

### 1️⃣ Загрузка фото (Device A)

```javascript
Пользователь выбирает фото → Браузер кодирует в Base64 
    → Сохраняет локально (localStorage) 
    → Готовит JSON для GitHub
    → Загружает на GitHub Pages
```

### 2️⃣ Синхронизация (Device B)

```javascript
Браузер проверяет GitHub каждые 5 сек 
    → Получает anna-photos-data.json 
    → Сравнивает с локальными фото 
    → Добавляет новые фото
    → Обновляет галерею
```

## Частые вопросы о синхронизации

### ❓ Как часто происходит синхронизация?
**Ответ:** Каждые 5 секунд автоматически, плюс при переключении вкладок/окон.

### ❓ Нужен ли интернет?
**Ответ:** Да, для синхронизации нужно подключение к интернету.

### ❓ Где хранятся мои фото?
**Ответ:** 
- Локально: в localStorage браузера (~5MB)
- В облаке: в файле anna-photos-data.json на GitHub

### ❓ Какой размер фото можно загружать?
**Ответ:** До 5MB, но рекомендуется сжимать большие файлы.

### ❓ Будут ли фото видны другим людям?
**Ответ:** Нет! Репозиторий приватный, доступ только у вас.

### ❓ Что будет если отключить интернет?
**Ответ:** Фото остаются в браузере локально, синхронизация возобновится при подключении.

### ❓ Можно ли удалить фото со всех устройств?
**Ответ:** Да, удалите фото в админ панели - удаление синхронизируется на все устройства.

## Отладка синхронизации

### Как проверить статус синхронизации?

1. **В интерфейсе:**
   - Смотрите статус в навигационной панели вверху
   - ✓ Синхронизировано = все хорошо
   - ✗ Ошибка = проблема с подключением

2. **В консоли браузера (F12):**
   ```javascript
   // Проверить все фото
   console.log(JSON.parse(localStorage.getItem('anna_photos')))
   
   // Проверить ID устройства
   console.log(localStorage.getItem('device_id'))
   
   // Проверить последнюю синхронизацию
   console.log(localStorage.getItem('anna_last_sync'))
   ```

### Как очистить синхронизацию?

```javascript
// В консоли браузера (F12):

// Очистить локальные фото
localStorage.removeItem('anna_photos')

// Очистить ID устройства
localStorage.removeItem('device_id')

// Очистить дату синхронизации
localStorage.removeItem('anna_last_sync')

// Очистить все
localStorage.clear()
```

## Безопасность данных

### Как защищены мои фото?

1. **LocalStorage браузера:**
   - Шифруется браузером
   - Доступно только с того же домена
   - Локально хранится на устройстве

2. **GitHub хранилище:**
   - HTTPS соединение (зашифровано)
   - Доступно только вам
   - Резервные копии GitHub

3. **Транспортировка:**
   - Все данные передаются по HTTPS
   - Защищено от перехвата

## Технические детали

### Как работает синхронизация в коде?

```javascript
// Функция синхронизации
async function syncPhotos() {
    // 1. Загружаем удаленные фото
    const remotePhotos = await fetchPhotosFromGithub()
    
    // 2. Сравниваем с локальными
    const newPhotos = remotePhotos.filter(remote => 
        !allPhotos.some(local => local.id === remote.id)
    )
    
    // 3. Добавляем новые
    if (newPhotos.length > 0) {
        allPhotos = [...allPhotos, ...newPhotos]
        savePhotosToStorage()
        renderGallery()
    }
}

// Запускаем каждые 5 секунд
setInterval(() => syncPhotos(), 5000)
```
